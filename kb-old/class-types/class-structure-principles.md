# Class Structure Principles

## Definition
Class structure is treated as a delivery system that must be deliberate, repeatable, and consistent if technical curriculum is to produce stable results.

## Key Details
- Learning quality depends on motivation, challenge, and fun as much as technical content.
- Coaches are expected to maintain a firm but not rigid environment.
- Assistant coaches are used to preserve class flow and to support struggling or disruptive students.
- Positive reinforcement is a preferred behavioral tool.

## Gaps
- The sources provide principles and examples rather than a single unchangeable SOP for every age group.
