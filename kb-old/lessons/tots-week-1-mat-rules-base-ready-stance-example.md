# Tots Week 1 Mat Rules, Base, And Ready Stance Example

## Definition
This page records a tots example lesson-format artifact for a short foundation-cycle class.

## Example Status
- This page preserves an example lesson format artifact and should not be treated as a canonical approved lesson.

## Structural Observations
- The example uses a simple lesson script rather than multi-option technical blocks.
- The example keeps movement, game, and intro-grappling sections distinct.
- The example presents the class as a short, sequential script from introduction through closing.

## Preserved Example Format

````markdown
 Example lesson plan format for tots:

```markdown
## Week 1 (Foundation)
Cycle: foundation
Theme: Mat Rules, Base, And Ready Stance
Movement Theme: Athletic base and stopping on command.
Class Length: 30 minutes

## Lesson
### Introduce the theme
Mat Rules, Base, And Ready Stance. Tell the class the goal is Mat Rules, Base, And Ready Stance.

### Warm-Up Block
- Run a movement-focused warm-up around Athletic base and stopping on command. and keep the pace playful.
- For example: Bear-crawls with stop and sit command.

### Main Activity Block
- Play Red light, green light with BJJ stance freezes. and use short resets to reinforce Balance, listening, and changing levels safely..

### Intro Grappling Block
- Expose the students to Basic stance, bow-in habits, and staying in personal space. with simple cues and lots of praise.

### Closing Script
- Recap the mat rules, reinforce the best example from the room, and set the expectation for live application.
```
````
