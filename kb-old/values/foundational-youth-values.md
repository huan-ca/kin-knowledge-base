# Foundational Youth Values

## Definition
The youth program’s value system is explicit and is meant to shape both training behavior and off-mat growth.

## Key Details
- Named values include respect, discipline, perseverance, humility, restraint, and loyalty.
- The lesson-plan guide expands values coaching into specific life-skills conversations such as adversity, integrity, excellence, and helping others.
- Value instruction is intended to be visible inside normal class operations, not separated from training.

## Gaps
- The source does not define a formal rubric for assessing value development beyond general coaching judgment.
