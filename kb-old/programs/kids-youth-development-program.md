# Kids Youth Development Program

## Definition
The youth program is framed as a character-building and life-development program that uses BJJ as its main vehicle rather than treating classes as pure technique delivery.

## Key Details
- The source targets physical, mental, and emotional development.
- The program aims to produce future stewards of both the art and the community.
- Values development is positioned as part of core curriculum quality, not an optional add-on.

## Gaps
- The source does not define a full program-level progression map across all ages and cohorts.
