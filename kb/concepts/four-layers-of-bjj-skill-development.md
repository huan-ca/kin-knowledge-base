# Four Layers of BJJ Skill Development

## Definition
The four-layer model describes BJJ development through techniques, muscle memory, principles, and intuition rather than treating progress as technique accumulation alone.

## Key Details
- Techniques provide the first clear recipe-like map for solving a problem.
- Muscle memory turns a known technique into something the body can execute under resistance.
- Principles explain why techniques work and help students troubleshoot and adapt.
- Intuition allows faster real-time decisions without conscious step-by-step analysis.
- The source argues that the strongest development happens when all four layers are trained together.

## Gaps
- The source presents a conceptual model, but it does not prescribe exact programming ratios or lesson allocations for each layer.
