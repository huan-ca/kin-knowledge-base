# Kids Youth Development Program

## Definition
The youth program is framed as a character-building and life-development program that uses BJJ as its main vehicle rather than treating classes as pure technique delivery.

## Detailed Notes
- The sources explicitly frame the youth program as more than technical instruction. Physical, mental, and emotional development are all part of the stated mission.
- BJJ is the main vehicle, but the intended outcome includes resilience, discipline, community behavior, and long-term stewardship of the academy culture.
- Values development is not described as a side conversation that happens only at promotions or speeches. It is treated as part of the real curriculum quality of the room.
- The program also assumes that technical structure, recognition, class standards, and life-skills conversations all contribute to whether the room actually produces those outcomes.

## Operational Implications
- Coaches should not evaluate class quality only by how many techniques were shown. Room culture, effort, attention, and behavior are part of the program’s success criteria.
- This page also supports keeping life-skills chats and recognition moments inside the main class flow rather than treating them as distractions.

## Gaps
- The source does not define a full program-level progression map across all ages and cohorts.
