# Foundational Youth Values

## Definition
The youth program’s value system is explicit and is meant to shape both training behavior and off-mat growth.

## Detailed Notes
- Named values in the raw material include respect, discipline, perseverance, humility, restraint, and loyalty.
- The kids lesson structure source broadens this into regular life-skills conversations around adversity, positive mindset, helping others, striving for excellence, authenticity, integrity, and showing respect.
- The important point is that values coaching is not meant to be abstract branding. It should show up in how the room runs, what coaches praise, what behavior gets corrected, and what examples are highlighted publicly.

## Operational Implications
- Coaches can reinforce these values during recognition moments, discipline corrections, life-skills chats, and even technical drilling behavior.
- This lets the values operate as observable training culture rather than only as poster language.

## Gaps
- The source does not define a formal rubric for assessing value development beyond general coaching judgment.
