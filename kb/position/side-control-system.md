# Side Control System

## Definition
Side control is presented as a stable control family with multiple variants built around chest pressure, upper-body isolation, and transition options.

## Detailed Notes
- The source treats side control as a family, not a single static pin. That matters because different frames, body types, and reactions call for different top-control solutions.
- Named variants include traditional cross-face and underhook, both-arms-far-side control, overhook and near-side-hip control, cross-face with near-side hip control, kesa gatame, and reverse kesa gatame.
- The compendium repeatedly organizes these pins through secure, attack, and advance lenses. In other words, the position is judged by how well it stabilizes the opponent, what attacks it offers, and what transitions it opens.
- Side control is also presented as a transition hub for mount, north-south, back exposure, and shoulder-focused attacks.

## Operational Implications
- Coaches can teach side control in modules rather than as one finished position: first stability, then transitions between variants, then attacks.
- This also aligns well with the example lesson format where a takedown lands into side control before the class moves into a submission block.

## Gaps
- The source offers rich content, but not yet a gym-approved age progression for each side-control variant.
