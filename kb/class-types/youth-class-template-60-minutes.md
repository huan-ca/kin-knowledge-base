# Youth 60-Minute Class Template

## Definition
The youth class template is a repeatable 60-minute structure used as the default delivery model for ages 8 to 13.

## Detailed Notes
- The class opens with bow in, announcements, and welcoming students.
- Warm ups are short and topic-aware.
- Takedown instruction and drilling come before ground instruction and drilling.
- Positional or open sparring takes up roughly the final third of class.
- Closing includes bow out and student recognition.
- Lesson segments should stay short, with transition time kept tightly controlled.

## Gaps
- The template is described as a rough guide, so coaches still need judgment for pacing and group management.
