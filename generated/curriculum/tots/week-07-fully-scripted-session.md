# Tots Week 07 Fully Scripted Session

## Opening Script
Introduce the theme: Level Changes And Penetration Steps. Tell the class the goal is Level Changes And Penetration Steps.

## Warm-Up Block
Run a movement-focused warm-up around Dropping levels and stepping forward without falling over. and keep the pace playful.

## Main Activity Block
Play Cone shot game with targets and quick resets. and use short resets to reinforce Knee bend, posture, and stepping patterns..

## Intro Grappling Block
Expose the students to Intro takedown footwork without hard finishes. with simple cues and lots of praise.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
