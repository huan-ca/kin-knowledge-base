# Tots Week 06 Fully Scripted Session

## Opening Script
Introduce the theme: Balance, Posture, And Push-Pull Games. Tell the class the goal is Balance, Posture, And Push-Pull Games.

## Warm-Up Block
Run a movement-focused warm-up around Staying tall while resisting light pushes and pulls. and keep the pace playful.

## Main Activity Block
Play Sumo circle balance game with controlled contact. and use short resets to reinforce Center of gravity, posture, and using feet to stay stable..

## Intro Grappling Block
Expose the students to How a grappler keeps balance when someone makes contact. with simple cues and lots of praise.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
