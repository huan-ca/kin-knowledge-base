# Tots Week 08 Fully Scripted Session

## Opening Script
Introduce the theme: Stand-Up Base And Technical Stand-Up Play. Tell the class the goal is Stand-Up Base And Technical Stand-Up Play.

## Warm-Up Block
Run a movement-focused warm-up around Posting, standing safely, and recovering base. and keep the pace playful.

## Main Activity Block
Play Treasure grab technical stand-up relay. and use short resets to reinforce Posting hand placement and standing without crossing feet..

## Intro Grappling Block
Expose the students to Technical stand-up as a movement skill and safety habit. with simple cues and lots of praise.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
