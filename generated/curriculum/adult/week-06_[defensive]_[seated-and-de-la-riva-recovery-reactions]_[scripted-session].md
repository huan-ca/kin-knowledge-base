# Adult Week 06 Fully Scripted Session

## Opening Script
Introduce the theme: Seated And De La Riva Recovery Reactions. Tell the class the goal is Use seated and De La Riva reactions to restore usable guard structure after grips break.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Recovering seated guard and De La Riva hooks after the passer clears the first line.
Teach Level 1 as the base pattern: Rebuild frames and reconnect a seated or De La Riva entry.
Layer Level 2 only after the room is stable: Use a second hook, collar grip, or angle change to off-balance the passer before recovering.
Repeat the coach emphasis: Adults should feel how hook placement changes the passer’s posture.

## Adult-Specific Notes
Keep the room grounded in guard recovery and off-balance basics before discussing more advanced inversions.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
