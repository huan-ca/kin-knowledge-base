# Adult Week 23 Fully Scripted Session

## Opening Script
Introduce the theme: Lower-Body Offense: Entries, Ashi Position, And Controlled Finishes. Tell the class the goal is Introduce lower-body submission offense through safe entries, control, and pressure management.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Safe leg-entanglement entries, ashi positioning, and controlled straight-footlock style finishing mechanics.
Teach Level 1 as the base pattern: Enter one stable lower-body control and maintain it safely.
Layer Level 2 only after the room is stable: Chain the first entry to a second control or follow-up finish when the defender rotates.
Repeat the coach emphasis: Control the knee line and partner safety before adding finishing pressure.

## Adult-Specific Notes
Mention IBJJF legality, but explain that the academy wants students aware of broader rule sets and defensive realities.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
