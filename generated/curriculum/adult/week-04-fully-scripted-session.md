# Adult Week 04 Fully Scripted Session

## Metadata
- Week: 04
- Cycle: offensive
- Theme: Closed Guard Submission Chains
- Teaching Goal: Create one reliable attacking sequence from closed guard with clean submission mechanics.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Closed Guard Submission Chains. Tell the class the goal is Create one reliable attacking sequence from closed guard with clean submission mechanics.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Closed guard armbar and triangle pathway with controlled finishing details.
Teach Level 1 as the base pattern: Attack the straight armbar or triangle after clearing posture.
Layer Level 2 only after the room is stable: Chain armbar to triangle or back to posture-breaking if the first attack stalls.
Repeat the coach emphasis: Frame the chain as posture break, angle, isolate, finish.

## Adult-Specific Notes
Mention the IBJJF safety lens, but also explain how the same mechanics show up across broader rule sets.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
