# Youth Week 03 Fully Scripted Session

## Opening Script
Introduce the theme: Closed Guard Control And Simple Sweeps. Tell the class the goal is Introduce closed guard as a controlled offensive platform with posture breaking and simple sweep entries.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Closed guard posture breaking and hip-bump style sweep threats.
Teach Level 1 as the base pattern: Break posture and connect a hip-bump sweep or sit-up sweep entry.
Layer Level 2 only after the room is stable: Use the sweep threat to force a post and connect to a second attack.
Repeat the coach emphasis: Reward angle creation over muscling the partner.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
