# Youth Week 22 Fully Scripted Session

## Opening Script
Introduce the theme: Choke Defense, Tap Awareness, And Reset Reactions. Tell the class the goal is Finish the defensive cycle by reinforcing hand fighting, neck safety, and calm reset behavior under submission pressure.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Defending rear strangles, cross-collar style threats, and recovering composure after a failed defense.
Teach Level 1 as the base pattern: Use hand fighting and shoulder alignment to remove immediate choke danger.
Layer Level 2 only after the room is stable: Chain the defense into a positional escape instead of only surviving the hands.
Repeat the coach emphasis: Make tap awareness and partner communication explicit.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
