# Youth Week 17 Fully Scripted Session

## Opening Script
Introduce the theme: Grip Fighting, Stance, And Sprawl Reactions. Tell the class the goal is Build standing defense through posture, inside position, and first-contact reactions.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Grip fighting, stance discipline, level-change awareness, and sprawls.
Teach Level 1 as the base pattern: Win inside position and sprawl safely against a basic shot.
Layer Level 2 only after the room is stable: Add a front-headlock or go-behind reaction after the sprawl succeeds.
Repeat the coach emphasis: Keep the head above the hips and teach students to feel pressure changes.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
