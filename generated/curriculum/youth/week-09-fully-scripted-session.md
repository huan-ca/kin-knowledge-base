# Youth Week 09 Fully Scripted Session

## Opening Script
Introduce the theme: Side Control Escape Layers And Kesa Survival. Tell the class the goal is Prepare students for heavy side control variations, including kesa and reverse-kesa style pressure.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Escapes from side control, kesa gatame, and transition pressure.
Teach Level 1 as the base pattern: Recover frames, turn to a hip, and re-guard.
Layer Level 2 only after the room is stable: Use a second-stage underhook or backdoor reaction when the top player chases the far side.
Repeat the coach emphasis: Students should know where their near-side elbow and knee belong at all times.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
