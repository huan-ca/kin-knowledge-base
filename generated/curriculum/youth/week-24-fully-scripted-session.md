# Youth Week 24 Fully Scripted Session

## Metadata
- Week: 24
- Cycle: self-defense
- Theme: Ground Self-Defense: Pins, Escapes, And Technical Stand-Up
- Teaching Goal: Finish the curriculum by connecting ground safety, pin survival, technical stand-up, and disengagement decisions.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Ground Self-Defense: Pins, Escapes, And Technical Stand-Up. Tell the class the goal is Finish the curriculum by connecting ground safety, pin survival, technical stand-up, and disengagement decisions.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Ground self-defense priorities: protect, frame, escape, stand, and create space.
Teach Level 1 as the base pattern: Recover frames, escape a basic pin, and stand up safely.
Layer Level 2 only after the room is stable: Add a second escape layer or controlled reversal before the stand-up when needed.
Repeat the coach emphasis: Keep the final action practical: get safe and regain distance.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
