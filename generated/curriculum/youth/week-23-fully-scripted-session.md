# Youth Week 23 Fully Scripted Session

## Opening Script
Introduce the theme: Standing Self-Defense: Distance, Base, And Clinch Safety. Tell the class the goal is Introduce standing self-defense through posture, distance management, breakfalls, and simple clinch exits.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Standing self-defense reactions, posture, breakfalls, and safe disengagement.
Teach Level 1 as the base pattern: Use stance, hands, and voice to create space and recover balance.
Layer Level 2 only after the room is stable: Add a clinch tie-up, basic off-balance, or technical exit when distance collapses.
Repeat the coach emphasis: Self-defense is not competition; the priority is safe escape and control.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
