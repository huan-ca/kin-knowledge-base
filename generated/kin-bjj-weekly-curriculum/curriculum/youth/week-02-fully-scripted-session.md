# Youth Week 02 Fully Scripted Session

## Metadata
- Week: 02
- Cycle: defensive
- Theme: Late Escape Chains And Safe Guard Recovery
- Teaching Goal: Teach students how to recover when the first escape fails and pressure is already established.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Late Escape Chains And Safe Guard Recovery. Tell the class the goal is Teach students how to recover when the first escape fails and pressure is already established.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Middle and late-stage escape chains from mount, knee on belly, and heavy side control

- **Primary Focus**
  - Coaching tip: "Survival details stay small and always connect back to guard recovery"
  - Level 1: Use elbow-knee movement and hip recovery to rebuild guard
  - Level 2: Add an ankle scoop or turn-in reaction when the top player overcommits
  - Outcome: Teach students how to recover when the first escape fails and pressure is already established

- **CLA/Situational Rolling Options**:
  - Start from the core position in Middle and late-stage escape chains from mount, knee on belly, and heavy side control and work until use elbow-knee movement and hip recovery to rebuild guard or the position resets.
  - Start from a successful first exchange and continue until add an ankle scoop or turn-in reaction when the top player overcommits or the defender escapes.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
