# Youth Week 10 Fully Scripted Session

## Metadata
- Week: 10
- Cycle: defensive
- Theme: Mount And Knee-On-Belly Escape Chains
- Teaching Goal: Link mount escapes and knee-on-belly escapes into repeatable reactions instead of isolated moves.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Mount And Knee-On-Belly Escape Chains. Tell the class the goal is Link mount escapes and knee-on-belly escapes into repeatable reactions instead of isolated moves.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Defensive reactions from mount and knee on belly into half guard or full guard recovery

- **Primary Focus**
  - Coaching tip: "Keep the chin safe and make students solve the pressure with structure first"
  - Level 1: Bridge, frame, and recover a safe half guard
  - Level 2: Chain bridge and elbow-knee reactions when the top player floats between mount and knee on belly
  - Outcome: Link mount escapes and knee-on-belly escapes into repeatable reactions instead of isolated moves

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
