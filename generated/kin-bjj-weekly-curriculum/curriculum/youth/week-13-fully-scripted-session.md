# Youth Week 13 Fully Scripted Session

## Metadata
- Week: 13
- Cycle: defensive
- Theme: Turtle Posture, Sit-Outs, And Front-Headlock Defense
- Teaching Goal: Use turtle as a dynamic defensive station instead of a hiding position.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Turtle Posture, Sit-Outs, And Front-Headlock Defense. Tell the class the goal is Use turtle as a dynamic defensive station instead of a hiding position.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Turtle structure, sit-outs, and defensive reactions to front-headlock pressure.
Teach Level 1 as the base pattern: Build turtle posture and escape to a safer neutral position.
Layer Level 2 only after the room is stable: Add a timed sit-out or stand-up when the top player reaches too far.
Repeat the coach emphasis: Protect the neck first, then move with purpose.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
