# Youth Week 01 Fully Scripted Session

## Metadata
- Week: 01
- Cycle: defensive
- Theme: Early Escape Frames From Mount And Side Control
- Teaching Goal: Build calm survival habits, frames, and first-movement escapes from major pins.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Early Escape Frames From Mount And Side Control. Tell the class the goal is Build calm survival habits, frames, and first-movement escapes from major pins.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Early-stage escape structure from mount and side control

- **Primary Focus**
  - Coaching tip: "Rule of three: frame, move hips, recover space"
  - Level 1: Bridge, frame, and recover half guard or closed guard
  - Level 2: Chain the first bridge into a second direction change when the top player posts
  - Outcome: Build calm survival habits, frames, and first-movement escapes from major pins

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
