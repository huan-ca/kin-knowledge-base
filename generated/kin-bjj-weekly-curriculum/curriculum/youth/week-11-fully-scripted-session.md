# Youth Week 11 Fully Scripted Session

## Metadata
- Week: 11
- Cycle: offensive
- Theme: Side Control Control To Americana
- Teaching Goal: Teach top players how to secure side control before attacking one high-percentage youth-safe submission.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Side Control Control To Americana. Tell the class the goal is Teach top players how to secure side control before attacking one high-percentage youth-safe submission.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Side control pressure, far-side isolation, and americana mechanics

- **Ground: Side Control (Top)**
  - Coaching tip: "Use shoulder pressure to pin rather than squeezing with the arms"
  - Level 1: Secure side control and finish the americana with clean elbow positioning

- **Submission: Americana**
  - Level 2: Flow from the americana threat into a straight-arm or mount transition when the partner defends
  - Outcome: Teach top players how to secure side control before attacking one high-percentage youth-safe submission

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
