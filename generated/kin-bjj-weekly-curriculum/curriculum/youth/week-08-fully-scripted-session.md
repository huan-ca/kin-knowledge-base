# Youth Week 08 Fully Scripted Session

## Metadata
- Week: 08
- Cycle: offensive
- Theme: Toreando And Leg-Drag Passing Chains
- Teaching Goal: Expand offensive passing into mobile routes that still finish with real control.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Toreando And Leg-Drag Passing Chains. Tell the class the goal is Expand offensive passing into mobile routes that still finish with real control.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Passing around the legs with toreando and leg-drag concepts

- **Primary Focus**
  - Coaching tip: "Do not let speed remove chest connection at the end"
  - Level 1: Use angle and grips to clear the feet and secure side control
  - Level 2: Switch from toreando to leg drag when the defender squares back in
  - Outcome: Expand offensive passing into mobile routes that still finish with real control

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
