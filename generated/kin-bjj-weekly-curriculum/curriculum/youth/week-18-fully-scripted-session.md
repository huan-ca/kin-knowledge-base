# Youth Week 18 Fully Scripted Session

## Metadata
- Week: 18
- Cycle: defensive
- Theme: Takedown Counters To Safe Top Entries
- Teaching Goal: Move from pure takedown defense into safe top-position conversions.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Takedown Counters To Safe Top Entries. Tell the class the goal is Move from pure takedown defense into safe top-position conversions.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Countering common takedown attempts and landing in stable top control

- **Takedown**
  - Coaching tip: "Reward balance and angle, not forcing the counter"
  - Level 1: Defend the attack and circle to a safe go-behind
  - Outcome: Move from pure takedown defense into safe top-position conversions

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
