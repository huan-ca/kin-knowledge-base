# Youth Week 23 Fully Scripted Session

## Metadata
- Week: 23
- Cycle: self-defense
- Theme: Standing Self-Defense: Distance, Base, And Clinch Safety
- Teaching Goal: Introduce standing self-defense through posture, distance management, breakfalls, and simple clinch exits.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Standing Self-Defense: Distance, Base, And Clinch Safety. Tell the class the goal is Introduce standing self-defense through posture, distance management, breakfalls, and simple clinch exits.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Standing self-defense reactions, posture, breakfalls, and safe disengagement

- **Primary Focus**
  - Coaching tip: "Self-defense is not competition; the priority is safe escape and control"
  - Level 1: Use stance, hands, and voice to create space and recover balance
  - Level 2: Add a clinch tie-up, basic off-balance, or technical exit when distance collapses
  - Outcome: Introduce standing self-defense through posture, distance management, breakfalls, and simple clinch exits

- **CLA/Situational Rolling Options**:
  - Start from the core position in Standing self-defense reactions, posture, breakfalls, and safe disengagement and work until use stance, hands, and voice to create space and recover balance or the position resets.
  - Start from a successful first exchange and continue until add a clinch tie-up, basic off-balance, or technical exit when distance collapses or the defender escapes.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
