# Youth Week 03 Fully Scripted Session

## Metadata
- Week: 03
- Cycle: offensive
- Theme: Closed Guard Control And Simple Sweeps
- Teaching Goal: Introduce closed guard as a controlled offensive platform with posture breaking and simple sweep entries.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Closed Guard Control And Simple Sweeps. Tell the class the goal is Introduce closed guard as a controlled offensive platform with posture breaking and simple sweep entries.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Closed guard posture breaking and hip-bump style sweep threats

- **Primary Focus**
  - Coaching tip: "Reward angle creation over muscling the partner"
  - Level 1: Break posture and connect a hip-bump sweep or sit-up sweep entry
  - Level 2: Use the sweep threat to force a post and connect to a second attack
  - Outcome: Introduce closed guard as a controlled offensive platform with posture breaking and simple sweep entries

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
