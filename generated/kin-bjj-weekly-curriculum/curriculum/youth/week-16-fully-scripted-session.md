# Youth Week 16 Fully Scripted Session

## Metadata
- Week: 16
- Cycle: offensive
- Theme: North-South Control And Transition Attacks
- Teaching Goal: Show students how north-south can be a controlling transition hub rather than just a pass-through position.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: North-South Control And Transition Attacks. Tell the class the goal is Show students how north-south can be a controlling transition hub rather than just a pass-through position.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: North-south control, spin transitions, and submission entry timing

- **Ground: Side Control (Top)**
  - Coaching tip: "Teach pressure and angle together; do not let students hover above the chest"
  - Level 1: Stabilize north-south and connect to one clean attack route

- **Submission: Kimura**
  - Level 2: Chain north-south to side control or kimura-style attack windows if the first route closes
  - Outcome: Show students how north-south can be a controlling transition hub rather than just a pass-through position

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
