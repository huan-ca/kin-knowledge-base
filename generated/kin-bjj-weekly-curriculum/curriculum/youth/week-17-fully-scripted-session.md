# Youth Week 17 Fully Scripted Session

## Metadata
- Week: 17
- Cycle: defensive
- Theme: Grip Fighting, Stance, And Sprawl Reactions
- Teaching Goal: Build standing defense through posture, inside position, and first-contact reactions.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Grip Fighting, Stance, And Sprawl Reactions. Tell the class the goal is Build standing defense through posture, inside position, and first-contact reactions.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Grip fighting, stance discipline, level-change awareness, and sprawls

- **Takedown**
  - Coaching tip: "Keep the head above the hips and teach students to feel pressure changes"
  - Level 1: Win inside position and sprawl safely against a basic shot
  - Outcome: Build standing defense through posture, inside position, and first-contact reactions

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
