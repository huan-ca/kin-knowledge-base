# Youth Week 15 Fully Scripted Session

## Metadata
- Week: 15
- Cycle: offensive
- Theme: Rear Mount Control To Strangle Finish
- Teaching Goal: Teach secure rear control and one disciplined strangle pathway that fits youth safety rules.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Rear Mount Control To Strangle Finish. Tell the class the goal is Teach secure rear control and one disciplined strangle pathway that fits youth safety rules.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Rear mount control, hook management, and clean rear-strangle mechanics

- **Primary Focus**
  - Coaching tip: "Keep the finish calm and immediate on the tap"
  - Level 1: Stabilize rear mount and complete the finish with proper hand placement
  - Level 2: Switch sides or trap a defensive hand before re-entering the finish
  - Outcome: Teach secure rear control and one disciplined strangle pathway that fits youth safety rules

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
