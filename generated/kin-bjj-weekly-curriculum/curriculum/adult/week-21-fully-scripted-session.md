# Adult Week 21 Fully Scripted Session

## Metadata
- Week: 21
- Cycle: defensive
- Theme: Armbar And Triangle Defense
- Teaching Goal: Use posture and angle awareness to survive and unwind common upper-body submission threats.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Armbar And Triangle Defense. Tell the class the goal is Use posture and angle awareness to survive and unwind common upper-body submission threats.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Defensive posture and escape routes against armbar and triangle attempts.
Teach Level 1 as the base pattern: Recognize the threat early and use one clean posture or stack escape.
Layer Level 2 only after the room is stable: Chain the first defense into a safe pass or neutral reset before the attacker reconnects.
Repeat the coach emphasis: Submission defense starts before the position is fully locked.

## Adult-Specific Notes
Tie the defense to safe passing or reset behavior rather than leaving the exchange neutral.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
