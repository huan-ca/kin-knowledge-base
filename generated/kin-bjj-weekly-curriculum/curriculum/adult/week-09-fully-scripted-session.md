# Adult Week 09 Fully Scripted Session

## Metadata
- Week: 09
- Cycle: defensive
- Theme: Side Control Escape Layers And Kesa Survival
- Teaching Goal: Handle side control pressure and scarf-hold variants without overexposing the back or arms.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Side Control Escape Layers And Kesa Survival. Tell the class the goal is Handle side control pressure and scarf-hold variants without overexposing the back or arms.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Escapes from side control, kesa gatame, and transition pressure

- **Primary Focus**
  - Coaching tip: "Students should understand which frame fails first under pressure and why"
  - Level 1: Recover frames, turn to a hip, and re-guard
  - Level 2: Use a second-stage underhook or backdoor reaction when the top player chases the far side
  - Outcome: Handle side control pressure and scarf-hold variants without overexposing the back or arms

## Adult-Specific Notes
Allow more weight and chest connection from the top player than in youth classes.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
