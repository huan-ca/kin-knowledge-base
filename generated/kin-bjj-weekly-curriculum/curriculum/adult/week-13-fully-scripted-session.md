# Adult Week 13 Fully Scripted Session

## Metadata
- Week: 13
- Cycle: defensive
- Theme: Turtle Posture, Sit-Outs, And Front-Headlock Defense
- Teaching Goal: Use turtle defensively without letting it become a dead-end shell.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Turtle Posture, Sit-Outs, And Front-Headlock Defense. Tell the class the goal is Use turtle defensively without letting it become a dead-end shell.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Turtle structure, sit-outs, and defensive reactions to front-headlock pressure

- **Primary Focus**
  - Coaching tip: "Compact hips and neck protection come first"
  - Level 1: Build turtle posture and escape to a safer neutral position
  - Level 2: Add a timed sit-out or stand-up when the top player reaches too far
  - Outcome: Use turtle defensively without letting it become a dead-end shell

## Adult-Specific Notes
Give adults more freedom to wrestle up or build front-headlock counters once the basic defensive structure is stable.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
