# Adult Week 02 Fully Scripted Session

## Metadata
- Week: 02
- Cycle: defensive
- Theme: Late Escape Chains And Safe Guard Recovery
- Teaching Goal: Prepare adults to recover from failed first escapes and restore guard under realistic top pressure.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Late Escape Chains And Safe Guard Recovery. Tell the class the goal is Prepare adults to recover from failed first escapes and restore guard under realistic top pressure.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
Show the core idea first: Middle and late-stage escape chains from mount, knee on belly, and heavy side control.
Teach Level 1 as the base pattern: Use elbow-knee movement and hip recovery to rebuild guard.
Layer Level 2 only after the room is stable: Add an ankle scoop or turn-in reaction when the top player overcommits.
Repeat the coach emphasis: Late escapes should still end in a structurally sound guard, not a scramble.

## Adult-Specific Notes
Allow the top player to float between mount and knee-on-belly so the bottom player must read transitions.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
