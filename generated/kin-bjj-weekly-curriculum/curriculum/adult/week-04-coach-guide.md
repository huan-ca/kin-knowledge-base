# Adult Week 04 Coach Guide

## Metadata
- Week: 04
- Cycle: offensive
- Theme: Closed Guard Submission Chains
- Teaching Goal: Create one reliable attacking sequence from closed guard with clean submission mechanics.
- Class Length: 60 minutes

## Weekly Intent
Create one reliable attacking sequence from closed guard with clean submission mechanics.

## Coaching Emphasis
Teach adults to make the attacking angle undeniable before extending the finish.

## Safety Notes
Match resistance to student readiness, reinforce tapping and partner care, and scale pace before complexity.

## Scaling Options
- Level 1 should focus on core mechanics and one clean finish or escape path.
- Level 2 should add chain options, timing, or more live reactions without changing the theme.

## Adult-Specific Notes
Mention the IBJJF safety lens, but also explain how the same mechanics show up across broader rule sets.
