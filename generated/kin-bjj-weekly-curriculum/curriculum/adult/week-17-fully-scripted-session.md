# Adult Week 17 Fully Scripted Session

## Metadata
- Week: 17
- Cycle: defensive
- Theme: Grip Fighting, Stance, And Sprawl Reactions
- Teaching Goal: Use standing defense to build confidence against pressure and takedown attempts.
- Class Length: 60 minutes

## Opening Script
Introduce the theme: Grip Fighting, Stance, And Sprawl Reactions. Tell the class the goal is Use standing defense to build confidence against pressure and takedown attempts.

## Warm-Up Block
Use a brief, topic-aware warm-up and set the expectation that the class will build one connected theme.

## Primary Focus Script
### Lesson: Grip fighting, stance discipline, level-change awareness, and sprawls

- **Takedown**
  - Coaching tip: "The stance should recover immediately after each defense"
  - Level 1: Win inside position and sprawl safely against a basic shot
  - Outcome: Use standing defense to build confidence against pressure and takedown attempts

- **CLA/Situational Rolling Options**:
  - Start from the core position in Grip fighting, stance discipline, level-change awareness, and sprawls and work until win inside position and sprawl safely against a basic shot or the position resets.
  - Start from a successful first exchange and continue until add a front-headlock or go-behind reaction after the sprawl succeeds or the defender escapes.

## Adult-Specific Notes
Bring in more hand-fighting detail and posture breaking than in the youth version.

## Closing Script
Recap the main idea, reinforce the best example from the room, and set the expectation for live application.
